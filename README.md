# 🚀 SmartHire AI - Resume Screening & Job Recommendation System

SmartHire AI is an AI-powered Resume Screening and Job Recommendation System that helps students find suitable jobs based on their skills and helps recruiters analyze candidates using skill matching and AI-based scoring.

## 📌 Project Overview

The system allows users to create profiles, upload resumes, search jobs, apply for jobs, and view AI-generated match scores. Recruiters can view applicants and shortlist or reject candidates.

## ✨ Features

### 👨‍🎓 Student Module
- ✅ Student Registration & Login
- ✅ User Profile Management

- ✅ Skill Extraction
- ✅ AI Resume Matching
- ✅ Job Search
- ✅ Apply for Jobs
- ✅ View Applications
- ✅ Match Percentage Calculation
- ✅ Missing Skills Analysis

### 👨‍💼 Recruiter Module
- ✅ Recruiter Dashboard
- ✅ View Candidate Details
- ✅ View AI Match Score
- ✅ Shortlist Candidates
- ✅ Reject Candidates
